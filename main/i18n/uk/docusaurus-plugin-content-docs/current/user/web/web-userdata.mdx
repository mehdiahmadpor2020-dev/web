## Огляд {#overview}

В інтернеті користувачі можуть отримувати доступ, керувати та створювати свої особисті дані, такі як Треки та Улюблені місця, безпосередньо в вебінтерфейсі.

- Треки: Користувачі можуть завантажувати, переглядати, редагувати або створювати нові треки онлайн. Усі зміни автоматично синхронізуються з OsmAnd Cloud, забезпечуючи доступність даних на всіх підключених пристроях.
- Улюблені місця: Створюйте та впорядковуйте збережені місця в інтернеті з автоматичною синхронізацією з вашим додатком OsmAnd для швидкого доступу та навігації.

Інтеграція з OsmAnd Cloud забезпечує безперебійну синхронізацію, дозволяючи користувачам зручно керувати своїми даними з будь-якого пристрою.

## Треки в інтернеті {#tracks-on-the-web}

Це платна функція <ProFeature/>. Спочатку вам потрібно увійти у свій обліковий запис OsmAnd Pro:

![Вхід для треків](@site/static/img/web/track_loggin.png)

Для розділу Треки:

- Показати треки з [OsmAnd Cloud](#cloud-tracks).
- Додавання треків на мапу (папка **Видимі на мапі**).
- Перегляд всієї інформації про треки та графіків.
- Змінювати треки та додавати їх до Хмари.
- Завантажувати та видаляти треки.
- Створювати нові папки або видаляти їх.
- Завантажувати папки як колекцію OSF або OBF.


### Видимі на мапі {#visible-on-the-map}

Тут знаходиться список треків, видимих на мапі.

_Вмикайте/вимикайте_ треки, щоб показати або приховати їх на мапі.

![Видимі на мапі](@site/static/img/web/visible.png)


### Меню папки треків {#track-folder-menu}

Натисніть кнопку *Три крапки* **⋮**, щоб відкрити меню *Папка треків*. Тут ви можете завантажити всі треки з обраної папки як:

 - Завантажити як OSF.
 - Завантажити як колекцію OBF:
  Ви можете отримати мапу OBF з цих даних: [Файл OBF](https://osmand.net/docs/technical/osmand-file-formats/osmand-obf/) або [Подорож OBF](https://osmand.net/blog/routes#generated-travel-routes).

    -  [Файл OBF](https://osmand.net/docs/technical/osmand-file-formats/osmand-obf/):

  Ви можете завантажити офлайн-мапу OBF (OsmAnd Binary Format) і відкрити її за допомогою OsmAnd на пристрої, це підходить для відображення великої кількості треків, які будуть видимі на мапі.

    -  [Подорож OBF](https://osmand.net/blog/routes#generated-travel-routes):

  Також ви можете імпортувати мапу треків як Книгу подорожей, що дозволить вам також вибирати конкретний трек на мапі та використовувати його як звичайний GPX. Книга подорожей також підтримує такі функції, як відображення треків у вигляді точок, фільтрація треків за типом активності та фільтрація шляхових точок.

- _Перейменувати_ обрану папку.
- _Видалити_ обрану папку.

![Меню папки треків](@site/static/img/web/collection.png)


### Хмарні треки {#cloud-tracks}

Треки GPX, які ви маєте в [OsmAnd Cloud](../personal/osmand-cloud.md), будуть доступні для відображення та редагування після входу в систему. Тільки для **користувачів Pro** <ProFeature/> та для користувачів [**OsmAnd Start**](../personal/osmand-cloud.md#osmand-start) (які можуть завантажувати свої дані навіть після закінчення терміну дії підписки Pro).

Наступні функції доступні після вибору хмарного треку:

- Інформаційні екрани:
  - *Інформація* - відображення даних треку.
  - *Висота* - графік висот.
  - *Швидкість* - графік швидкості.
  - *Нахил* - графік нахилу.
  - *Налаштування* - список профілів навігації для інструменту *Створити маршрут*.  

- Кнопки дій:
  - *Редагувати трек* - додавання хмарного треку до локального сховища для редагування (кнопка _Редагувати локально_ на панелі).
  - *Завантажити GPX* - завантажує трек на ПК.
  - *Перерахувати* висоту (Супутник) - відображає графік висот вибраного треку.
  - *Видалити трек* - видаляє трек.
  - *Закрити трек* - закриває трек.

![Редагування GPX у хмарі OsmAnd Web](@site/static/img/web/cloud_track.png)



## Улюблені місця в інтернеті {#favorites-on-the-web}

Після реєстрації [**OsmAnd Pro**](../personal/osmand-cloud.md#login) та для [**OsmAnd Start**](../personal/osmand-cloud.md#osmand-start), ваші Улюблені місця відображаються в меню Улюблені місця.  
Доступні наступні дії:

- *Показати на мапі* - показати улюблені точки з обраної папки на мапі.
- *Перейменувати* - назва та опис папки улюблених місць.
- *Завантажити* - завантажити обрану папку улюблених місць.
- *Видалити* - видалити обрану папку улюблених місць.
